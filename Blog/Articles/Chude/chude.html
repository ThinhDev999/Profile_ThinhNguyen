What Is Unity?

December 7, 2023

Unity is a cross-platform game engine developed by Unity Technologies, mainly used to develop video games for 21 platforms including computers, game consoles (like PlayStation) and phones, etc.

More than 50% of the games on the market are produced by Unity. Some extremely famous games created by Unity can be mentioned as Pokémon Go, Hearthstone, Ori And The Blind Forest, Monument Valley, Axie Infinity,... Unity's "coverage" is very wide, it can be Commonly applied in many different game genres from "heavyweight" Triple A (AAA) games to simple educational games for children.

What are the basic features of Unity?

Unity strongly supports the following features: Create the Game's UI interface such as creating a drop bar, textbox,... Display 3D, 2D models – 2D, 3D physical systems Networking: Supports creating multiplayer games at the same time New special platform support: AR – Augmented reality, VR – Virtual reality AI – Supports bots in the game screen, supports packages to create bots in the game Supports displaying text with special fonts … Of all Unity's features, Huy personally tops the Editor application interface. Unity Editor, with many tools, supports game development tasks such as creating 3D models (physics, lighting) and writing scripts,... just by dragging and dropping images. Thanks to this feature, Unity supports other positions in the game development process such as Game Designer.
